 # Amazon Price Tracker Bot 🛒

A Python automation bot that scrapes Amazon product prices daily and notifies you via email when the price drops below your target.

### ✨ Current Features - Day 1
- Scrapes live product title and price from Amazon
- Uses headers to avoid getting blocked  
- Clean error handling and status logs
- Built with Python + BeautifulSoup + Requests

### 🔧 Tech Stack
`Python` `BeautifulSoup4` `Requests` `SMTP` `SQLite` - coming soon

### 🚀 Roadmap
- **Day 2:** Add email alert system using Gmail SMTP
- **Day 3:** Store price history in SQLite database
- **Day 4:** Add support for Daraz, eBay, AliExpress
- **Day 5:** Build simple dashboard to view price trends

### 📦 How to Run
1. `pip install requests beautifulsoup4`
2. `python main.py`
